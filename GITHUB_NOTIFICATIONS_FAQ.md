# GitHub Notifications on iPhone - FAQ

## How do I get my notifications to show as badges for GitHub on iPhone?

To enable notification badges for GitHub on your iPhone, follow these steps:

### 1. Enable Notifications in the GitHub Mobile App

1. Open the **GitHub mobile app** on your iPhone
2. Tap your profile picture in the top right corner
3. Tap **Settings** (gear icon)
4. Tap **Notifications**
5. Enable the types of notifications you want to receive:
   - **Participating** - Notifications for conversations you're participating in
   - **Watching** - Notifications for repositories you're watching
   - **Pull requests** - Notifications for pull request reviews
   - **Issues** - Notifications for issues assigned to you or mentioned

### 2. Enable Notifications in iOS Settings

1. Open **Settings** on your iPhone
2. Scroll down and tap **GitHub**
3. Tap **Notifications**
4. Enable **Allow Notifications**
5. Make sure **Badges** is turned ON
6. You can also enable:
   - **Sounds** - For notification sounds
   - **Banners** - For notification banners
   - **Lock Screen** - To show notifications on lock screen

### 3. Check Your GitHub Web Settings

1. Go to https://github.com/settings/notifications on your browser
2. Under "Notification delivery," ensure **Mobile** is enabled
3. Configure which events trigger notifications

### Troubleshooting

If badges still aren't showing:

- **Check Do Not Disturb**: Make sure Do Not Disturb mode is off, or that GitHub is allowed during Do Not Disturb
- **Check Focus Modes**: If you use Focus modes, ensure GitHub notifications are allowed
- **Reinstall the app**: Sometimes reinstalling the GitHub app can fix badge issues
- **Check iOS version**: Make sure your iOS is up to date
- **Background App Refresh**: Enable Background App Refresh for GitHub in Settings > General > Background App Refresh

### Additional Tips

- Badge count shows the number of unread notifications
- You can customize notification settings per repository on GitHub.com
- Clearing notifications in the app will also clear the badge count
- You can mute repositories or conversations you don't want to receive notifications for
